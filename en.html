KeepinCRM Smart UI Styler (Chrome Extension)

2. Purpose of the Extension

This extension customizes the KeepinCRM card interface on *.keepincrm.com domains (deals, clients, tasks). Corporate users authenticate through the JustSchool backend to enable personalized UI templates and real-time features.

3. Data We Collect

The extension does not collect URLs of websites outside *.keepincrm.com in the active-tab field of heartbeat payloads.

4. Where Data Is Sent

Authentication, heartbeat, and telemetry data are transmitted exclusively to the JustSchool corporate backend to operate the UI tuner. Endpoints:

  • https://webhooks.justschool.com.ua/webhook/v1/keepintuner (HTTPS)
  • wss://webhooks.justschool.com.ua/justhook/v1/platform/keepintuner/ws (WSS) — real-time events such as opening a deal

Data is used solely to provide extension functionality. It is not sold to third parties and is not used for advertising.

5. Security and Retention

  • Personal data in transit is encrypted via HTTPS and WSS.
  • JWT tokens and UI settings are stored locally in the user’s Chrome profile.
  • Server-side retention follows JustSchool internal policy; account data can be deleted on request.

6. Incognito Mode

The extension uses incognito: spanning. Authentication state and UI settings are shared between regular and Incognito windows within the same Chrome profile, so the tuner works consistently in both modes.

7. Your Rights

You may:

  • Sign out in the extension popup (removes the local JWT).
  • Uninstall the extension (removes local extension data).
  • Contact draboleg233@gmail.com to request deletion of server-side account data.
